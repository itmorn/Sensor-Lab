## 介绍
这里是硬件的“游乐场”与“档案馆”。我是一个沉迷于拆解与收藏的算法工程师。

在这里，我用工程师的视角，带你从原理到代码，系统性地玩转每一个传感器、执行器和通信模块。这不是简单的教程堆砌，而是一个有“收集癖”的开发者，为你亲手搭建的、可随时查阅的硬件模块博物馆。

电路的实验沙盘，无需焊接，用杜邦线在孔阵间搭建临时电路。这里是验证想法最快捷的试验场，是硬件开发者最忠实的草稿纸：

- 左右两列+-常用做电源和接地，纵向全部联通

## 二极管
二极管是电子世界里最基础、最经典的非线性元件之一。它就像电路的“交通警察”，只允许电流单向通行，这种看似简单的特性，却衍生出整流、保护、稳压、发光等千变万化的应用。
