## 本征半导体
本征半导体是所有半导体技术与器件的理论起点和性能参照基准。其纯净结构下热激发产生电子-空穴对的行为，是理解掺杂、PN结及晶体管工作原理的根本基础。

- 概念：是一种纯净的具有晶体结构的半导体
- 常用：硅、锗。这些四价原子最外层有4个电子，原子之间能形成共价键，从而达到一个稳定的结构
    - ![alt text](img/20260107101139.png)
- 晶体结构
    - 像东方明珠，正四面体
    - ![alt text](img/20260107101619.png)
- 载流子
    - 包含：自由电子和空穴。
    - 本征激发：只要不是绝对零度，电子就会存在热运动，就可能挣脱束缚成为自由电子。
    - 空穴：是共价键中电子挣脱后留下的、带正电的空缺。它的运动本质是相邻电子的依次填补。等效为一个具有正电荷的载流子。
    - ![alt text](img/20260107113153.png)

## 杂质半导体
杂质半导体是通过向本征半导体中掺入特定杂质，从而人为控制其导电类型（N型/P型）和导电能力的关键材料，是所有半导体器件的物理基础。

- 概念：掺入**少量**杂质元素，少量不会改变晶体结构
- N型半导体：掺入少量磷（P）
    - ![alt text](img/20260107113207.png)
    - 多子：多数载流子，这里是自由电子
    - 少子：少数载流子，这里是空穴
- 杂质半导体（N型/P型）的类型：是根据其**多数载流子**的电荷性质来定义的。
    - N型（Negative Type）：多数载流子是带负电荷（Negative）的自由电子。
    - P型（Positive Type）：多数载流子是带正电荷（Positive）的空穴。
- 温度影响：温度升高会加剧本征激发，使载流子总数增加。
    - 多数载流子：在常用温度范围内，其浓度由掺杂决定，基本保持稳定。
    - 少数载流子：浓度随温度呈指数上升，是影响器件高温性能的关键。
- 施主原子：贡献载流子的原子
- P型半导体：掺入少量硼（B），引入空穴作为多数载流子。
    - 多子：空穴
    - 少子：自由电子

## PN结
PN结是“Positive-Negative Junction”的简称，指P型与N型半导体结合形成的交界区域。当两者结合时，在交界面处会形成一个具有单向导电性、宽度稳定的空间电荷区（耗尽层），这就是PN结，它是所有半导体器件的核心结构。


![alt text](img/20260107144641.png)

PN结的“平衡”是指 **扩散作用** 和 **漂移作用** 达到动态平衡，可以这样理解这个过程：
1. 扩散开始：
    - 当P型半导体（多空穴）和N型半导体（多电子）刚结合时，P区的空穴会扩散到N区，N区的电子会扩散到P区。
    - 这会在交界处形成一个带电的 “空间电荷区”（也叫耗尽层），P区一侧带负电，N区一侧带正电。
2. 内建电场形成：
    - 这个空间电荷区会产生一个从N区指向P区的内建电场（也叫势垒电场），会阻碍多数载流子的进一步扩散。
3. 产生漂移作用：
    - 这个内建电场会推动带负电的电子向N区运动（漂移），推动带正电的空穴向P区运动（漂移）。
    - 漂移运动的方向正好和之前的扩散运动方向相反。
4. 达到平衡：
    - 开始时，扩散作用强，漂移作用弱。但随着扩散的进行，空间电荷区变宽，内建电场增强，扩散作用变弱，漂移作用变强。
    - 最终，由浓度差驱动的扩散运动 和 由内建电场驱动的漂移运动达到平衡状态，空间电荷区的宽度不再变化。

- 一句话总结：
    - PN结的平衡是“扩散”与“漂移”两种运动力量相等、相互抵消的动态平衡状态。

- 对称结
    - P区和N区掺杂浓度相近。
    - 空间电荷区向两侧延伸宽度相近。

- 不对称结（单边突变结）
    - 一侧浓度远高于另一侧（如P⁺N或PN⁺）。
    - 空间电荷区几乎全部向低浓度一侧延伸，高浓度一侧很薄。（绝大多数实际PN结属于此类） 

- 单向导电性
    - 加正向电压
        - ![alt text](img/20260107162655.png)
        - 外加电场会削弱内建电场，从而实现导通
        - PN结的导通电流和外部电压**不是线性关系**，外部电压很小时，几乎不导通，称为**死区**，随着电压的加大，通过的电流会指数级增大。
        - PN结内部电阻很小，一旦导通，电流会趋于无限大，热功率P=I*U，发生热击穿，所以要加一个限流电阻。
    - 加反向电压
        - ![alt text](img/20260107164816.png)
        - PN结变厚，阻挡作用更强，几乎不导电

- PN结的伏安特性
    - ![alt text](img/20260107165827.png)
    - 正向：
        - 电压从0开始加，有一个“死区”（硅管约0~0.5V），电流极小；超过约0.7V后，电流像开闸洪水一样指数级暴增。
    - 反向：
        - 电压反着加，电流始终极小（几乎为零），像个绝缘体；但电压加得过高时，会瞬间“击穿”，电流急剧增大。
        - 雪崩击穿：
            - 高电压把载流子“发射”出去，像子弹一样撞出更多载流子，引发雪崩式倍增。

        - 齐纳击穿：
            - 超强电场直接把半导体原子上的电子“撕”下来，瞬间制造大量自由载流子。
            - 应用：稳压二极管

## 二极管
二极管是一种基本的非线性元件，其核心特性是单向导电性，这使得它在整流、保护、稳压和发光等电路中发挥关键作用。与线性电阻不同，二极管的电压与电流关系不符合欧姆定律，因此需要深入理解其伏安特性和导通规律，才能进行准确的电路分析和设计。

- 常见结构
    - ![alt text](img/20260108110106.png)
    - （a）：点接触型，金属丝是P
    - （b）：面接触型
    - （c）：平面型，集成芯片基础单元

- PN结和二极管的关系
    - PN结：空间电荷区
    - 二极管：
        - 包含：P区、N区、PN结、P区体电阻、N区体电阻、外壳封装、外壳泄露电流

- 二极管的伏安特性
    - ![alt text](img/20260108111203.png)
    - 相同电压下电流比PN结小：由于体电阻的存在
    - 反向电流大：由于外壳泄漏电流
    - 温度升高，增加参与导电的载流子数量，电流增大（室温下，温度提升1度，正向导通电压会减小约 2 mV ~ 2.5 mV；每升高10度，反向电流增加一倍）。

- 二极管的应用
    - 利用正向导通特性：稳压、钳制电位
    - 利用反向饱和电流特性：温度传感器
    - 利用反向击穿特性：稳压二极管（正向电压受限于材料（约0.7V），反向电压可通过掺杂自由设计（几伏到几百伏），因此更灵活。）

- 二极管的主要参数
    - $I_F$（最大平均整流电流）：二极管长期运行时允许通过的最大正向平均电流。超过此值会因过热而损坏。
    - $U_R$（最大反向工作电压）：允许施加在二极管两端的最大反向电压。通常为反向击穿电压 $U_{BR}$ 的 1/2，以确保安全裕量。
    - $I_R$（反向饱和电流）：二极管加反向电压且未击穿时流过的微小电流。
    - $f_M$（最高工作频率）：由 PN结电容 和 反向恢复时间 决定。超过此频率，二极管将无法正常实现单向导电性。
    - $U_F$（正向压降）：二极管通过额定正向电流 $I_F$ 时两端的电压降。硅管典型值约 0.6~0.8V。
    - $C_J$（结电容）：PN结形成的电容，影响高频性能。
    - $t_{rr}$（反向恢复时间）：二极管从正向导通切换到反向截止所需的时间，对开关速度至关重要。

## 发光二极管
发光二极管，简称LED，是一种能将电能直接转化为光能的半导体器件。其核心原理是电子与空穴在耗尽区附近复合时（电子和空穴在这里才能大量相遇），以可见光子形式释放能量，从而实现高效发光。

![alt text](img/20260108171631.png)

- 发光原理
    - ![alt text](img/20260108165803.png)
    - 当电子从导带落入价带的空穴时，该带隙能量就会以光子形式发射
    - 材质不同，光色不同：砷化镓（GaAs）发红外光，磷化镓（GaP）发红光/绿光，氮化镓（GaN）发蓝光/紫外光
    - 普通半导体材质（如硅）发射不可见红外光，能量多以热形式释放

## 电容器
电容器是一种能存储和释放电荷的储能元件，通过两块导体间夹一层绝缘介质构成，具有隔直流、通交流的特性。

![alt text](img/20260113150811.png)

- 陶瓷电容器（扁豆电容器）
    - ![alt text](img/20260113150817.png)
    - ![alt text](img/20260113150826.png)
    - 两个金属圆盘 + 一个陶瓷圆盘作为电介质

- 电容器的原理
    - ![alt text](img/20260113150851.png)
    - 电源正极 → 从电容器的正极板持续抽出电子。
    - 电源负极 → 向电容器的负极板持续堆积电子。
    - 两极板上的异性电荷建立起一个从正极板指向负极板的均匀电场。
    - 当电容器两极板间的电压 U_c 上升到与电源电压 V 完全相等时，电荷停止移动，电流变为零。
    - 充电完成后，电容器像一个 “微型电池”，将能量以电场能的形式储存在两极板间的电场中。

- 电容的定义式和决定式
    - 电容的定义式
        - $$C = \frac{Q}{U}$$
        - 含义：电容器所带的电荷量 Q 与两极板间的电势差 U 的比值，定义为该电容器的电容 C
        - 定义式**并没有揭示**电容 C 的大小由哪些物理因素决定。它只是告诉我们如何量化和计算电容。

    - 电容的决定式（以平行板电容器为例）
        - $$C = \frac{\varepsilon_r \varepsilon_0 S}{d}$$
        - S: 两极板的正对面积
        - d: 两极板间的距离
        - $ε_r$: 两极板间电介质的相对介电常数
        - $ε_0$: 真空介电常数
        - 这是一个决定式，它揭示了电容大小的内在决定因素。
        - 电容是电容器自身的属性，与它是否带电、带多少电、两端电压多大无关（类似于电阻是导体本身的属性，与是否通电无关）。


- 电容值的决定因素
    - **面积**：两极板正对的有效面积越大，能“盛放”的电荷越多，电容越大。
    - **距离**：两极板间的距离越近，正负电荷间的吸引力越强，在相同电压下能吸引更多的电荷，电容越大。
    - **电介质**：加上电介质后，它会减弱板上同种电荷的排斥，使更多电荷能聚集在板面，从而增大电容。

- 电容的极性
    - 电容分两种 —— 有极性的（电解电容）和没极性的（陶瓷、薄膜电容）。
    - 有极性电容的优势：用很小的体积，就能做到超大容量。代价就是必须注意正负极，不能接反。

## 电感器
电感器是一种利用通电导体周围产生磁场来储存磁场能量、并对电流变化产生阻碍作用的元件，具有“通直流、阻交流”的特性。

![alt text](img/20260113195458.png)

- 原理
    - 根据安培定律，当导体中有电流通过时，导体周围会产生磁场；线圈中的电流越大，产生的磁场越强。
        - ![alt text](img/20260113195506.png)
    - 当电流发生变化时，线圈中的磁通量随之变化，根据法拉第电磁感应定律，线圈中会产生感应电动势（自感现象）。
        - ![alt text](img/20260113195514.png)
    - 该感应电动势的方向总是阻碍原电流变化（楞次定律），从而表现为对电流变化的阻碍作用。
    - 电感器正是利用这种磁场储能和电磁感应效应来工作，其储存的能量以磁场能的形式存在。

- 简单电路
    - 刚接通电流无法通过电感
        - ![alt text](img/20260113195519.png)
    - 电感磁通量不在变化，通直流
        - ![alt text](img/20260113195527.png)
    - 断开电路，电流发生瞬间变化，磁通量继续变化，产生电动势给灯泡供电。
        - ![alt text](img/20260113195534.png)
